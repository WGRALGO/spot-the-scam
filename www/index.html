About

Spot the Scam is a free educational scam-awareness game created by WGRALGO / The Wealth Gap Resolution Algorithm™ Inc. It helps users practice spotting common red flags in scams involving money, identity, jobs, banking, delivery messages, fake prizes, tech support, and urgent payment demands.

This app works offline. It does not collect personal data, does not use ads, does not use analytics, does not use trackers, does not require an account, and does not upload game activity to any server.
